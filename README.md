# 伊藤忠100股計畫助手

這不是技術分析網站，而是協助長期累積伊藤忠（8001）至 100 股的個人投資助手。

核心用途：

- 防止衝動加碼。
- 追蹤持股進度。
- 記錄所有買進紀錄。
- 計算平均成本。
- 提供掛單建議。

## 使用方式

```bash
npm install
npm run dev
```

打開 Vite 顯示的 `Local:` 網址即可使用。此版不依賴 API，資料會存在瀏覽器的 localStorage，重新整理後仍會保留。

## 主要功能

- 持股總覽：股票名稱、股票代號、目前持股、目標持股、完成率、剩餘股數。
- 成本追蹤：總投入金額、平均成本、目前價格、未實現損益、損益百分比。
- 交易紀錄：新增、編輯、刪除買進紀錄。
- 今日掛單建議：小買、中買、大跌三段式建議，價格由目前價格動態計算。
- 今日評價：提醒可小買、觀望或不建議追價。
- 個人規則設定：可設定每一段的跌幅百分比與最大股數。
- 走勢圖：優先使用 API 取得日線收盤價，失敗時可手動匯入 CSV。
- 未來預留：API自動同步、NISA追蹤、月度投入統計、配息追蹤、投資日誌。

## 每日 30 秒流程

1. 輸入目前價格。
2. 輸入今日最多投入金額。
3. 看今日評價與追價警告。
4. 直接照小買 / 中買 / 大跌掛單。

## 預設掛單規則

- 小買：目前價格下跌 0.3%，最多 1 股。
- 中買：目前價格下跌 1.2%，最多 2 股。
- 大跌：目前價格下跌 3.0%，最多 5 股。

掛單價格會四捨五入到 0.5 円。

## CSV 匯入格式

交易紀錄：

```csv
日期,價格,股數,備註
2026-05-26,1920,10,初始建倉
2026-05-27,1918,10,加碼
2026-06-04,1850,1,NISA
```

走勢圖也支援：

```csv
date,open,high,low,close,volume
2026-06-01,1850,1870,1840,1860,1000000
2026-06-02,1860,1880,1830,1850,1200000
```

或只提供收盤價：

```csv
date,close
2026-06-01,1860
2026-06-02,1850
```
